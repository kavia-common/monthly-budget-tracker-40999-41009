# budget_tracking_frontend

Mobile app for tracking monthly budgets, transactions, and a dashboard with category breakdown.

- 4 tabs: Dashboard, Transactions, Budgets, Settings
- Ocean Professional theme (primary #2563EB, secondary/success #F59E0B, error #EF4444)
- Local in-memory seed data for categories, transactions, and budgets
- No backend required

Run:
- flutter run
- Or open in supported preview; app shows Dashboard by default.
